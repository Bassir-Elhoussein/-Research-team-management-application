== Description ==

Ayurvedic Medicine WordPress theme for ayurveda, herbal, body clinic, skin, spa treatment, health, wellness, naturopathy, homeopathy, healing, spa, massage, reiki, yoga, meditative practice, exercise, stretching, discipline, physical, spiritual, relaxing help, green, nature, indian medicinal herbs etc. It is responsive, user friendly, works well with Gutenberg editor and WooCommerce compatible and SEO plugins compatible. Flexible, scalable and easy to use.
